# RiGEL Ground Control System (GCS)

A portable and fully functional **UAV Ground Control System (GCS)** developed for Linux, supporting simulation, hardware integration, and seamless deployment.

---

## 🚀 Overview

RiGEL is a customized and portable Ground Control System designed to:

* Monitor and control UAVs
* Run multi-vehicle simulations
* Interface with real flight hardware
* Deploy across Linux systems without dependency issues

This project was developed as part of an internship focused on building a **complete UAV control and deployment ecosystem**.

---

## ✨ Key Features

* 🛰️ Mission planning and telemetry monitoring
* 🧪 Simulation using PX4 SITL + Gazebo
* 🔌 Hardware integration (Cube flight controllers, GPS modules)
* 📦 Portable deployment:

  * AppImage packaging
  * Self-extracting Linux installer
* 🔁 Multi-UAV simulation support
* 🔗 MAVLink-based communication

---

## 🛠️ Tech Stack

* **Languages/Tools:** C++, Python, Bash
* **Frameworks:** Qt (QML, QtQuick)
* **Simulation:** PX4 Autopilot, Gazebo
* **Communication:** MAVLink
* **Build Tools:** CMake, GCC, Ninja
* **Deployment:** AppImage, Shell scripting
